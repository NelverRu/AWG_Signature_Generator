## 📖 О проекте

**AmneziaWG Signature Generator** — инструмент для генерации аутентичных сигнатур клиента QUIC (I1–I5), которые маскируют WireGuard-трафик под обычные HTTPS-запросы на основе протокола **HTTP/3 (QUIC)**.

Это позволяет обходить DPI (Deep Packet Inspection) и цензуру, делая WireGuard неотличимым от стандартного веб-трафика.

> 🛡️ **Принцип работы:** Вместо стандартных handshake-пакетов WireGuard генерируются QUIC-пакеты с корректными тегами `I1`, `I2`, `I3`, `I4`, `I5`, имитируя реальное соединение с сервером (например, Google, Cloudflare или YouTube).

---

## ✨ Особенности

- 🎲 **Генерация I1–I5 сигнатур** — полный набор тегов инициализации QUIC-соединения.
- 🔄 **Реалистичные параметры** — правильные версии QUIC (Q043, Q046, v1), идентификаторы соединений и CID.
- 🧩 **Поддержка WireGuard** — генерация ключей и масок для встраивания в WireGuard-конфиги.
- 📁 **Экспорт в конфиги** — автоматическое создание готовых `.conf` файлов для клиента AmneziaWG.
- 🚀 **Высокая производительность** — написан на Go, минимальное потребление ресурсов.
